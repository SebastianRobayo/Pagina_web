Aprende sobre HTML

HTML (Lenguaje de Marcas de Hipertexto, del inglés HyperText Markup Language) es el componente más básico de la Web. HTML utiliza "marcas" para etiquetar texto, imágenes y otro contenido para mostrarlo en un navegador Web.

Etiqueta html

Está etiqueta es la que contendra todo nuestro proyecto y siempre una etiqueta tendra una de apertura y otra de cierre como se muestra en la imagen.


Etiqueta head

Está etiqueta sigue despues de la etiqueta html y contiene todos los elementos no visuales que ayudan a que la pagina funcione correctamente y se escribe de la siguiente manera.


Etiqueta body

Está etiqueta sigue despues de la etiqueta head todos los elementos visuales-estructurales estan contenidos dentro de la etiqueta body y se escribe de la siguiente manera.

Etiqueta title

Está etiqueta va dentro de la etiqueta head y es para colocarle un titulo a la pestaña de tu pagina y se escribe de la siguiente manera.


Etiqueta para los encabezados

HTML incluya seis niveles de etiquetas para en cabezados, los cuales estan clasificados segun su importancia y las cuales van dentro de las etiquetas body.


Etiqueta para realizar lineas horizontales

Está etiqueta es usada para realizar lineas horizontales en el documento.
